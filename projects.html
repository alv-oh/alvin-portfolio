Things I've built, end to end.

AirtimeX — paused

AirtimeX is a full-stack fintech mobile app that lets Safaricom users convert spare airtime into M-Pesa mobile money. The project is currently paused while I focus on coursework, but the core architecture is built and the basic conversion flow works in testing.

I built the full stack: React Native frontend using Expo, a Node.js and Express backend, PostgreSQL database handling users and transaction history, and JWT authentication. The Daraja API sandbox integration for the actual airtime-to-M-Pesa conversion is in place and working in test mode.

What's left before it's production-ready: proper error handling for partial transaction failures, a retry queue for failed M-Pesa callbacks, and real device testing beyond my own phone. I learned the hard way that anything touching money needs rollback logic before it touches real users — that's the main thing this project has taught me so far.

Tech Stack

  • React Native (Expo) — frontend, built into an Android APK
  • Node.js / Express — backend API
  • PostgreSQL — database
  • JWT — authentication
  • Safaricom Daraja API — airtime-to-M-Pesa conversion
  • Railway — deployment
