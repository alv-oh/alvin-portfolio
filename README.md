# Alvin's Personal Portfolio & Technical Showcase

## Project Overview

This production-ready web application serves as my professional portfolio. As a BBIT student at Strathmore University, I am passionate about bridging technology with real-world solutions. This site showcases my journey, detailing projects like my flagship application "AirtimeX", my ongoing research into AI/ML, and my literary exploration of African culture.

## Architecture & Tech Stack

- **Semantic HTML5:** Built with a focus on web accessibility (ARIA standards).
- **Modular CSS3:** Custom design system utilizing variables to extend and override Bootstrap defaults.
- **JavaScript (ES6+):** Modular interactivity including dark mode toggling, form validation, and content filtering.
- **Bootstrap 5:** Fully responsive framework utilizing advanced grid systems, offcanvas navigation, and modals.
